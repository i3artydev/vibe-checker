# Vibe Checker
Free website security, SEO & performance audit tool.
50+ automated checks — no signup required.

🔗 Live: https://websitesecurityscanner.netlify.app

## Checks include
- HTTPS / SSL
- Security headers (CSP, HSTS, X-Frame-Options...)
- SEO tags, Open Graph, Twitter cards
- Schema.org / JSON-LD
- robots.txt & sitemap
- Cookie consent & privacy policy
- Core Web Vitals guidance
- And more...

Built with vanilla JS + React (no build step).
